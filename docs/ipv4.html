構成

  • ネットワークアドレスとホストアドレス

  • 192.168.10.20/24 = 192.168.10.20/255.255.255.0

    • 192.168.10.20
      • IP アドレス
    • / 以降
      • サブネットマスク(ネットマスク)
      • CIDR 表記
      • ビットで表現すると /24 は、左端から 1 が 24 個続き、残りは 0 になる

  • ネットワークアドレス

    • ホスト部のビットがすべて 0
    • ホストに割り当て不可

  • ブロードキャストアドレス

    • ホスト部のビットがすべて 1
    • ホストに割り当て不可
