PKI(Public Key Infrastructure : 公開鍵基盤)

  • 公開鍵暗号技術を使った電子証明書を中心とした社会基盤
  • PKI では公開鍵の所有者を第三者信用機関(TTP)に保証してもらう
    • 第三者信用機関は公開鍵の所有者を何らかの方法で確認して、公開鍵とその所有者を保証する電子証明書を発行
    • 電子証明書は公開鍵と所有者の情報が記載された上で、TTP が署名を行う
    • この証明書を発行する機関のことを認証局(CA)
