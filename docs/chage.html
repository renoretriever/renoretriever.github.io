chage

  • ユーザパスワードの変更設定

  • ユーザの確認

    # chage -l test01
Last password change                                    : Apr 08, 2014
Password expires                                        : never
Password inactive                                       : never
Account expires                                         : never
Minimum number of days between password change          : 0
Maximum number of days between password change          : 99999
Number of days of warning before password expires       : 7
    項目説明
    Last password changeパスワード最終更新日
    Password expiresパスワードの有効期限
    never は無期限
    Password inactiveパスワードが無効になってから、本アカウントが無効になるまでの日数
    Account expiresアカウントが無効になる日
    Minimum number of days between password changeパスワード変更が向こうな期間
    0 はいつでも変更OK
    Maximum number of days between password change有効期限
    99999は無期限
    Number of days of warning before password expires有効期限きれる何日前に警告を出すか
