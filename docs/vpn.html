VPN(Virtual Private Network)

  • 盗聴、改ざんなどの脅威にさらされているパブリックネットワーク上に、暗号化技術などを用いて仮想的なプライベートネットワークを実現する技術の総称

種類

インターネット VPN

  • インターネットを介して接続

  • メリット

    • 通信回線のコストを抑えることができる

  • デメリット

    • パブリックインターネットなので、通信速度は不安定になる可能性がある

  • IPSec

    • AWS の VGW(バーチャルプライベートゲートウェイ)

  • SSL-VPN

    • 装置に必要なもの
      • FQDN または IP アドレスを含むディジタル証明書

IP-VPN

  • 通信事業者の IP 通信網を使用
  • メリット
    • 遅延時間の保証や稼働率の保証など、細かなサービスレベルを設定できる
  • デメリット
    • 高価

Others

  • カプセル化
    • 本来のパケット(ヘッダ情報を含むパケット) に新しいヘッダ情報を付加すること
    • トンネリング
      • カプセル化されたパケットを、そのプロトコルを使用できるネットワークを通じて送受信すること
  • 種類
    • インターネット VPN
    • IP-VPN
  • Others
