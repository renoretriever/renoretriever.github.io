chmod

  • パーミッションの変更

  • ファイルの所有者または、root しか実行できない

  • 注意点

    • ディレクトリが実行可能でない場合、その中にあるファイルのパーミッションに関係なく、一切アクセスできなくなる

  • 構文

    $ chmod [パーミッション] ［ファイル名]

  • パーミッション設定に使用できる数字

    itemcontents
    7オール OK
    6読み出し、書き込み
    5読み出し、実行
    4読み出し
    3書き込み、実行
    2書き込み
    1実行
    0オール NG

  • パーミッション設定に使用できる記号

    itemcontents
    u所有者
    gグループ
    oその他ユーザ
    aすべてのユーザ

  • 設定変更の内容

    itemcontents
    +許可を与える
    −許可を剥奪
    =新たに設定

  • 操作内容

    itemcontents
    r読み取り
    w書き込み
    x実行
