  • 代表的なものは HDD
  • RAID(Redundant Array of Independent Disks)

ネットワーク

  • ストレージをネットワークで接続する方法
    • NAS(Network Attached Storage)
      • ネットワークに接続して使用することのできるストレージ(ファイルシステム)のこと
    • SAN(Storage Area Network)
      • LAN とは別の専用ネットワークにより、ファイバチャネルスイッチ(FC スイッチ)と呼ばれる中継装置を介してストレージ装置を複数のサーバに接続し共有させるもの
      • SAN では、通常の LAN で用いる TCP/IP とは異なるファイバチャネル(FC)と呼ばれるプロトコルを用いており高速・効率的なデータ転送ができ、ストレージ装置の統合や LAN を介さずに SAN 経由でのデータバックアップが可能
      • 通常、サーバのシステムボードに FC ポートは付いていないため、PCI スロットに HBA と呼ばれるカードを挿す
    • DAFS(Direct Access File System)
      • データセンターやクラスタ環境でのファイルアクセスと管理のためのプロトコル
      • 多数のサーバからなる環境でも、ローカル・システムのように見える
    • DAS(Direct Attached Storage)
      • サーバとストレージを 1 対 1 で接続したもの

ディスク I/O ボトルネック

  • 対策
    • I/O の効率を上げる
    • I/O を減らす工夫をする
  • ディスク I/O 実施先
    1. サーバ内部(ローカルディスク)
    2. FC 経由での SAN ストレージ
    3. ネットワーク経由で NAS

  • シーケンシャル I/O

    • 先頭から間を抜かさずアクセス(読み/書き) すること

  • ランダム I/O

    • ヘッドを動かしながら飛び飛びにアクセス(読み/書き) すること
