json_verify

  • json フォーマットのチェック
  • サンプル

    • json

      # cat /tmp/sample.json
{
  "marker":[
    {"lat":41.772596,"lng":140.725261,"name":"函館朝市"},
    {"lat":41.796864,"lng":140.756965,"name":"五稜郭"},
    {"lat":41.772912,"lng":140.816231,"name":"函館空港"}
  ]
}

    • check(valid)

      # cat /tmp/sample.json | json_verify
JSON is valid
