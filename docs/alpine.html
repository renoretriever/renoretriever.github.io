alpine

  • インストール済みパッケージ一覧

    apk info

  • パッケージ更新

    apk update

  • インストール可能なパッケージ一覧

    apk search パッケージ名

  • インストール

    apk --update add パッケージ名

  • キャッシュの削除

    rm -rf /var/cache/apk/*

  • 特定バージョンのリポジトリを利用する

    • ここ からダウンロードする

    • リポジトリを書き換える(v3.6 に変更)

      $ cat /etc/apk/repositories
# cat /etc/apk/repositories
http://dl-cdn.alpinelinux.org/alpine/v3.6/main
http://dl-cdn.alpinelinux.org/alpine/v3.6/community
