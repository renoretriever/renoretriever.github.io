フィッシング詐欺

基礎

  • 行やオークションサイトを装った偽のホームページを開設し、偽のメールやリンク等で利用者を巧みに誘導してユーザ ID やパスワードを盗み出す行為

  • フィッシングサイト

  • 脆弱性と特徴

    • クロスサイトスクリプティング
    • SQL インジェクション
    • セキュリティホールを利用した攻撃
      • 攻撃手法
        • サイトの OS やアプリの脆弱性を悪用するコードを送信
      • 主な目的
        • サイトの利用機能を不正に入手し、踏み台に利用したり、ホームページの改ざんを行なう
      • サイトへのアクセス
        • サイトに対して直接アクセスして攻撃
    • フィッシング詐欺
      • 攻撃手法
        • 偽サイトを作成し、HTML メールやリンクで誘導
      • 主な目的
        • クレジットカード番号、口座番号、ユーザ ID、パスワード等を入手
      • サイトへのアクセス
        • 対象サイトとは無関係なところに偽サイトを設置するため、アクセス自体が発生せず検知が困難
  • 基礎
