コマンド

  • キュー数確認

    # /var/qmail/bin/qmail-qstat

  • キューヘッダ情報

    # /var/qmail/bin/qmail-qread

  • 添付ファイルのサイズを指定しているか下記を確認

    /var/qmail/control/databytes
    • デフォルトは0(無制限)

  • 設定内容確認

    # /var/qmail/bin/qmail-showctl

  • キューに滞留したメッセージの強制再送信

    $ kill -s ALRM “qmail-send のPID”

  • タイムアウト

    • リモートSMTPサーバーに接続する際のコネクションタイムアウト
      • /var/qmail/control/timeoutconnect
      • 秒数指定
      • デフォルト : 60秒
    • リモートSMTPサーバーからのレスポンスに対するタイムアウト
      • /var/qmail/control/timeoutremote
      • デフォルト : 1200秒
    • ローカルSMTPサーバーがリモートのSMTPクライアントからの応答に対するタイムアウト
      • /var/qmail/control/timeoutsmtpd
      • デフォルト : 1200秒

  • .qmail

    • 注意点
      • 空行もメールアドレスと判断されるので注意
        • ドメインが無い場合には、me の設定が踏襲される
  • プロセス
    • concurrencylocal
      • ローカル配送を処理するプロセスの上限
      • 上限は 120
    • concurrencyremote
      • リモート配送を処理するプロセスの上限
      • 上限は 120
