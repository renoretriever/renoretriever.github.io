vpopmail

  • バーチャルドメイン

  • キャッチオールの設定(.qmail-default)

    • 宛先不明の場合には転送を行う

      | /home/vpopmail/bin/vdelivermail '' test@example.com

運用

  • 仮想ドメイン追加

    • virtualdomain は、qmail-send にて読みこまれているので再起動が必要
    • HUP シグナルで設定ファイルの再読み込みが走るので、それでも良い

  • ex.com に test ユーザを追加する

    # /home/vpopmail/bin/vadduser test@ex.com

  • ユーザ情報確認

    # /home/vpopmail/bin/vuserinfo test@ex.com

  • ドメイン全体のユーザ情報確認

    # /home/vpopmail/bin/vuserinfo -D ex.com

移設

  • 基本的に /home/vpopmail 配下を rsync なりでコピーしてくるけど、vadddomain, vadduser コマンドでアカウントは事前に作成しておく必要がある
