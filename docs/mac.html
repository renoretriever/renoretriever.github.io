  • ホスト名の変更 for CLI

    • 既存の設定を確認

      • コンピュータ名

        $ scutil --get ComputerName

      • ローカルホスト名

        $ scutil --get LocalHostName

    • 各種名前を test に変更

      $ sudo scutil --set ComputerName test
$ sudo scutil --set LocalHostName test

  • ファイルサーバ(samba)への接続が遅い場合

    • カーネルパラメータ変更

      $ sudo sysctl -w net.inet.tcp.delayed_ack=0
net.inet.tcp.delayed_ack: 3 -> 0

    • 恒久反映

      $ sudo vim /etc/sysctl.conf
net.inet.tcp.delayed_ack=0

      • 新規作成の場合には下記を実施

        sudo touch /etc/sysctl.conf
sudo chown root:wheel /etc/sysctl.conf
sudo chmod 0644 /etc/sysctl.conf

      • 注意点としては、=の前後にはスペースは含めない

システム環境変数

  • 副ボタン
    • トラックパッド -> 副ボタンのクリックを右下隅をクリックに変更
  • アプリ内切り替え変更
    • ショートカット
    • Option + tab
  • space 設定で、アプリの利用度に応じて並び替えしないようにする
    • Mission Control -> 最新の使用状況に基いて操作スペースを自動的に並び替えるのチェックを外す
  • Dock 自動的に消す
    • Dock -> Dock を自動的に表示/非表示にチェック
  • 暗号化 ON
    • セキュリティとプライバシー -> FireVault をオンに
  • 輝度を自動調整させない
    • ディスプレイ
      • 内蔵 Retina ディスプレイ -> 輝度を自動調整のチェックを外す
    • キーボード
      • キーボード -> 環境光が暗い場合にキーボードの輝度を調整
  • キー速度(キーボード)
    • キーのリピートを速いに
    • リピート入力認識までの時間を短いに
  • 日付(日付と時刻)
    • 秒を表示
    • 24 時間表示にする
    • 日付を表示
