WebDAV(Web-based Distributed Authoring and Versioning)

  • HTTP を用いてファイルやフォルダの作成や削除、ファイル所有者や更新日時などの情報を取得・設定するだけでなく、クライアント側で作成したファイルを Web サーバにアップロードすることも可能
  • WebDAV を同じくファイル転送を行うプロトコル FTP と比較した場合、HTTP を拡張した仕様であるため、ブラウザ上から操作できることや SSL によるセキュアな通信コネクションを使用することができる

WebDAV で拡張させた概念

  • リソース

    • ファイルやコレクションのこと

  • コレクション

    • フォルダやディレクトリ内にあたる要素

  • プロパティ

    • リソースやコレクションが持つことが出来る追加の属性
    • 作成日時や更新日時、最終更新者といった情報が該当
    • 定義
      • 名前=値

  • ロック

    • 分散ファイルシステムでは、同じフォルダを複数人が同時に見てデータを共有できるが、同じファイルの同時更新を防ぐ仕組み
