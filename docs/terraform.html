AWS 環境構築

  2. コンフィグレーションファイルの作成

    • AWS Provider
    • 変数の持ち方
    • 認証情報

      • variables.tf

        provider "aws" {}

      • 実行

        $ export AWS_ACCESS_KEY_ID="anaccesskey"
$ export AWS_SECRET_ACCESS_KEY="asecretkey"
$ export AWS_DEFAULT_REGION="us-west-2"
$ terraform plan

  3. 初期化

    $ terraform init

  4. ドライラン

    $ terraform plan

  5. 適用

    $ terraform apply
