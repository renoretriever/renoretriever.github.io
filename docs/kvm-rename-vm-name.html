仮想マシンの名前を変更する方法

  • 前提

    • hoge => fuga に変更

  • 設定フィアルのコピー

    # cd /etc/libvirt/qemu
# cp -a hoge.xml hoge.xml.org
# cp -a hoge.xml fuga.xml

  • UUID を作成

    # uuidgen

  • 仮想イメージファイル名を変更

    # cd /var/lib/libvirt/images/
# mv hoge.qcow2 fuga.qcow2

  • 設定ファイルの変更

    # cd /etc/libvirt/qemu
# vim fuga.xml
    • 変更箇所
      • name
      • uuid
      • source file
      • channel の source mode

  • 変更前の仮想マシンの削除

    • これを行うと hoge.xml が削除される

      # virsh undefine hoge

  • 変更後の仮想マシンの登録

    # virsh define fuga
