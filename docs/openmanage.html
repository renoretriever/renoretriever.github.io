openmanage

  • DELL 製品システム管理ツール

  • ESM ログ

    # omreport system esmlog > esm.log

  • 警告ログ

    # omreport system alertlog > alert.log

  • システムログ

    # omreport chassis > chassis.log

  • ハードディスクログ

    • ディスクステータスを確認するにはまずコントローラー ID を確認

      # omreport storage controller

    • 仮想ディスク(Virtual Disk)

      # omreport storage vdisk controller=[上記で確認した ID] > vdisk.log

    • 物理ディスク(physical Disk)

      # omreport storage pdisk controller=[上記で確認した ID] > pdisk.log
      • pdisk で応答ない場合は、adisk へ変更

    • RAID コントローラバッテリログ

      # omreport storage battery controller=[上記で確認した ID] > battery.log

    • RAID コントローラログ

      # omconfig storage controller action=exportlog controller=[上記で確認した ID]
      • 上記のログの出力先は/var/log/lsi_xxxx.log(xxxxは実行日)
