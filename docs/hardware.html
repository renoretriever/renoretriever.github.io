ハードウェア

  • 入力装置
    • キーボード
    • マウス
  • 出力装置
    • ディスプレイ
    • プリンタ
  • 演算装置
    • CPU
  • 制御装置
    • CPU
  • 記憶装置
    • 主記憶装置
      • メモリー
    • 補助記憶装置
      • 種類
        • 磁気ディスク
          • ディスクの磁気の力を使って、データの書き込みする
        • 光ディスク
          • 薄い円盤にレーザを当てることによって、データを読み書きする
        • 半導体メモリ
          • 半導体素子によって構成されている
          • フラッシュメモリ
            • SSD
            • SD カード
            • USB メモリ
      • デフラグメンテーション
        • ユーザがデータの追加や削除を繰り返し行うと、記憶装置の中でデータがきれいに並ばず、飛び飛びの状態で保存された状態
  • バス
    • 上記装置同士をつなぐ回線
    • メモリマップド I/O
      • バスに接続された各装置には、CPU からアクセスするためのアドレス
    • 種類
      • データバス
        • 実際にデータを転送するときに利用
      • アドレスバス
        • アドレスを指示するときに利用
      • 制御バス
        • データを書き込むのか読み込むのかを指示するのに利用
    • バスは接続されているすべての機器で共有
      • => 同時に 2 種類のデータ転送は不可 、同じ時刻には、 2 つの装置間で片方向にしかデータをやり取りすることができない
      • => バスが空くのを待っている間はにどんどん届くデータは、いったん バッファ に格納され、バスが利用可能になるまで待たされる
      • => すべての機器でバスを共有するのではなく、複数のバスを用意し、間にコントローラスイッチを入れて、共有部分を減らす構造が主流
    • データ転送量(帯域、スループット)
      • 転送幅(一度にデータを送ることができるデータの幅) x 転送回数(1 秒間に何回送ることができるか)
        • 転送回数 = 1 秒 / 1 処理あたりの所要時間(レスポンス)
  • OCR
    • 印刷物や手書きのものをイメージスキャナなどで読み込む場合、基本的にはすべて画像データとして読み込まれれます。印刷物などに文字が含まれている場合に、文字を文字データとして読み込む技術や装置

コンピュータシステム

  • 磁気ディスク

    • 記憶容量
      • シリンダ数、1 シリンダ当たりのトラック数、1 トラック当たりの記憶容量
    • アクセス時間
      • 位置決め時間 + 回転待ち時間 + データ転送時間
    • 待ち時間
      • 位置決め時間 + 回転待ち時間
    • アクセス時間
      • 待ち時間 + データ転送時間

  • MIPS(Million Instructions Per Second)

    • プロセッサが 1 秒間に実行する命令の百万単位の数値
  • 割り込み
    • 内部割り込み
      • プログラムの命令を実行した結果として発生
      • ゼロによる助産
      • 演算結果のあふれ
      • 権限のないメモリ領域へのアクセス
      • オペランド語境界制限違反
      • 割り込み発生命令(スーパバイザコール)
    • 外部割り込み
      • 実行中のプログラムとの命令とは関係なく発生
      • 商用電源の瞬時停電などの電源異常による割り込み
      • 入出力が完了したことによる割り込み
      • メモリパリティエラーが発生したことによる割り込み
      • ウォッチドッグタイマ
        • システムの故障を検出するために時間を監視するタイマで、あらかじめ設定された時間になってもタイマが初期化されなかった場合、割り込みを用いてシステムの異常を通知する
        • ノンマスカブル割込み
  • キャッシュメモリ
    • CPU と主記憶とのデータのやり取りを高速化するためのメモリ
    • 実効メモリアクセス時間
      • キャッシュメモリのアクセス時間 * ヒット率 + 主記憶装置のアクセス時間 * (1 - ヒット率)
    • 割当方式
      • 主記憶と CPU の速度差を埋めるために置かれる高速のバッファ記憶
      • 主記憶上のメモリブロックと、キャッシュメモリ上のロケーションを対応させる方式
      • ダイレクトマッピング方式
        • メモリブロック毎にキャッシュメモリ上の特定の領域と対応付ける方式
      • フルアソシアティブ方式
        • メモリブロックをキャッシュ内の任意の領域に対応付ける方式
      • セットアソシアティブ方式
        • 主記憶のあるブロックをキャッシュメモリの複数の特定のブロックのまとまり(セット)に対応付ける方式

  • ライトプロテクト

    • フロッピーディスクや MO などの媒体で書込みや削除を禁止にする機構

  • ディスクキャッシュ

    • CPU とディスクの間で、データのやり取りを高速化する装置
