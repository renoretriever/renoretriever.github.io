IP アドレス

  • PC やサーバなど TCP/IP で通信する機器(ホスト)を識別するためのアドレス

  • IPv4

  • IPv6

  • グローバル IP アドレス

    • インターネット上で重複しない一意の IP アドレス
    • ICANN(Internet Corporation for Assigned Names and Numbers) にて管理
    • 家庭内で利用する場合には ISP から自動的に割り当てられる
      • => 接続する度にグローバル IP アドレスは変わる

  • プライベート IP アドレス

    • グローバル IP アドレスとしては利用されない範囲の IP アドレスであり、自由に利用することができる
    • 範囲
      • 10.0.0.0 〜 10.255.255.255(10.0.0.0/8)
      • 172.16.0.0 〜 172.31.255.255(172.16.0.0/12)
      • 192.168.0.0 〜 192.168.255.255(192.168.0.0/16)
    • NAT(Network Address Translation)
    • 同一ネットワーク内では一意にする必要がある

  • リンクローカルアドレス

    • ルータを介さず、ホスト間で直接通信するときに使える IP アドレス
    • 169.254.0.0 〜 169.254.255.255
    • ホストがネットに正しくつながっていない場合や、DHCP サーバと正しく通信できなかった場合に、設定されることがある

  • ループバックアドレス

    • 現在使用しているシステムを指すアドレス
    • 127.0.0.0 〜 127.255.255.255
    • 127.0.0.1 には localhost というホスト名が付けられている

  • マルチキャストアドレス

    • 特定のグループ間の複数のアドレス宛に同報通信を行うためのアドレス
    • 224.0.0.0 〜 239.255.255.255
    • IPv6 では IPv4 のマルチキャストとブロードキャストを兼ねた役割
