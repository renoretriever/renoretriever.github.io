SSD(Solid State Disk)

  • メモリと同じく半導体でできているが、不揮発性

HDD との違い

-HDDSSD
データ保存形式磁気的に保存電子的に保存
性能上の特徴物理的に連続したデータの読み書きが得意(まとめて読み書きするのが得意)
ランダムなデータの読み書きは苦手		ランダムなデータの読み書きが得意(HDD の 100 程度高速)
物理的に連続したデータの読み書きはあまり得意ではない(書き込みは HDD より遅いこともある)
信頼性長時間のデータ保存にも耐えうる
振動/衝撃に弱い		長時間のデータ保存に弱い
書き込みの多い環境では寿命が短命
振動/衝撃に強い
価格安価高価(DRAM よりは安価)
