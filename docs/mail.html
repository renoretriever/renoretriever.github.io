ヘッダー情報

  • ヘッダー情報とボディの間は空行で分ける

  • ヘッダFrom

    • 差出人に記載される情報
    • 実際に送信されるアドレスとは異なることもある

  • ヘッダTo

    • 宛先に記載される情報
    • 実際に送信されるアドレスとは異なることもある

  • エンベロープFrom

    • 実際の送信先などの情報が記載される

  • In-Reply-To

    • メールに対する返信であることを示す

迷惑メール対策

  • SPF
  • DKIM
  • ホスト名の逆引き
  • reCAPTCHA
  • tarpitting(ターピッティング)
    • スパムメールを排除するために、SMTP プロトコルの通信をわざと遅らせる手法
    • 応答をわざと遅らせることで、応答を送る前に相手が次のコマンドを送ってくるかを確認
      • 相手がすぐに次のコマンドを送ってくるようであれば、スパム送信者として受信を拒否
    • sendmail
      • グリーティングメッセージを送らせる greet pause という機能として実装
    • Postfix
      • 設定ファイル中に sleep 指定をすることで任意の応答を遅延させることができる
  • ブラックリスト確認
    • https://www.dnsbl.info/dnsbl-database-check.php
    • https://www.aguse.jp/
  • IP レピュテーションの確認
    • https://www.talosintelligence.com

メール送信のセキュリティ

  • Outbound Port 25 Blocking
  • SMTP Authentication (SMTP Auth)
    • SMTP でユーザ認証を行う技術
  • SMTP over SSL(SMTPS)
    • SSL/TLS を利用して SMTP 通信を暗号化する技術
    • 注意点
      • メールソフトとメールサーバが SMTPS に対応する必要がある

メール受信のセキュリティ

  • APOP
  • POP3 over SSL

メール本文のセキュリティ

  • PGP
  • S/MIME
