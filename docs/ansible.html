mysql_user モジュール

  • root ユーザで接続し、webapp ユーザを作成する

    - name: User add web app
  mysql_user:
    login_user: root
    login_password: 'CHANGEME'
    name: webapp
    host: '%'
    password: 'CHANGEME'
    priv: '*.*:ALL,GRANT'
    state: present

yum モジュール

  • state

    state意味対応するyumコマンド
    presentパッケージのインストール
    インストールされていれば skip    		yum install
    installedpresent の aliasyum install
    latestパッケージの最新版へのアップデートyum update
    absentパッケージのアンインストールyum remove
    removedabsent の aliasyum remove
