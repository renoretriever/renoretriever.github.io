sudo

  • 一般ユーザーに管理者コマンドの使用を許可するもの

  • /etc/sudoers

    • visudo の実行で上記ファイルの編集

    • 書式

      root    ALL=(ALL)       ALL
ユーザ名 ホスト名=(実行ユーザ名) [NOPASSWORD:] コマンド
      itemcontent
      ユーザ名コマンドの実行を許可するユーザ名か、グループ名か、もしくは ALL
      ホスト名実行を許可するホスト名か、IP アドレス、もしくは ALL
      実行ユーザ名コマンド実行時のユーザ(省略時は root)、もしくは ALL
      コマンド実行を許可するコマンドのパス、もしくは ALL
      NOPASSWD:指定すると、コマンド実行時にパスワードを問われない

  • /etc/sudoers.d 配下のファイル名には . をつけてはダメ

  • no password で sudo 利用

    • visudo で下記追加

      username ALL=NOPASSWD: ALL

  • 別ユーザ(test)でコマンドを実行

    $ sudo -u test ls

  • tty をもっていなくても、sudo 実行できる設定(visudo)

    Defaults !requiretty

    • ユーザ(admin)を指定

      Defaults:admin !requiretty
