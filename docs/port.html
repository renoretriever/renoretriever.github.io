ポート番号

  • サーバ上で稼働しているアプリケーションの識別を行うための技術
  • 16 ビットの数値

    • ポート番号は 0 〜 65535 の範囲

    • ウェルノウンポート

      • 0 〜 1023
      • IANA によって割り当てられている
      ProtocolTCPUDP
      HTTP80-
      HTTPS443-
      SMTP25-
      POP3110-
      DNS5353

    • 登録済ポート

      • 1024 〜 49151
      • よく利用されるアプリケーション用のポート
      • 8080 とか

    • ダイナミック/プライベートポート

      • 49152 〜 65535
      • 主に クライアントアプリケーション を識別するためのもの
      • クライアントアプリケーションが通信を行う際には、この範囲からランダムに割り当てる
