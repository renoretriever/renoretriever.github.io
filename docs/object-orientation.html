オブジェクト指向

  • データと手続きをひとまとまりにしたオブジェクトをベースにしてプログラムを構成していくプログラム言語
  • C++、 C#、Java などがオブジェクト指向型に分類
  • オブジェクトとは
    • なんでも(実体あり/なし)
    • 人間、犬、季節、思想 etc...
      • 概念も含む
    • 利用するプログラミンの中でオブジェクトを定義してあげる
  • 人間目線で考える

クラス

  • 共通する特徴をもつオブジェクトをまとめて定義したもの、分類

  • 例

    • 哺乳類
    • 爬虫類
    • 魚類

  • 人は哺乳類であり、生物でもある

    • 生物 の視点で見ると爬虫類も哺乳類も同じ class になる

  • 要素

    • クラスの名前
      • 例 : 人間
    • データ(属性)
      • 例 : 氏名、身長
    • 振る舞い(動作)
      • 例 : 走る、寝る

特徴

  • 継承
    • 具体化の段階において上位の性質を共有すること
  • カプセル化
    • データや機能の利用範囲を明確にする
      • 触られたくないデータは見せない
      • 変更したいデータは適切な機能を利用して実施する
        • アクセッサメソッド
  • ポリモーフィズム
    • 同じ機能でも機能内の動作に差が出ること
      • hello() メソッドを継承している各クラスを想定
        • こんいちはー、と返す
        • hello と返す
