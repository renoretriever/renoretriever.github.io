OSI 基本参照モデル 7 階層モデル

  • OSI(Open Systems Interconnection)
  • 現在は使われてないけど、参照モデルとして残っている
  • 国際標準化機構である ISO(International Organization for Standardization) が制定したもの

アプリケーション層

  • アプリケーションのやり取り
  • プロトコル
    • HTTP
    • SMTP

プレゼンテーション層

  • データの表現方法

セッション層

  • 通信の開始と修了の手順

トランスポート層

  • ネットワークの通信管理
  • プロトコル
    • TCP
    • UDP
  • 機器
    • ゲートウェイ(L4 以上)
      • 異なったプロトコル体系のネットワーク間の接続などに用いられる

ネットワーク層

  • ネットワークにおける通信経路の選択
  • プロトコル
    • ICMP
  • 機器
    • コリジョンドメイン
      • 分割する
    • ブロードキャストドメイン
      • 分割しない
    • 特徴
      • MAC アドレスを基にあて先を判断する
        • VLAN を利用することでブロードキャストドメインを分割できる
    • ブリッジ
      • コリジョンドメイン
        • 分割する
      • ブロードキャストドメイン
        • 分割しない
      • 特徴
        • フレームフィルタリング機能
        • MAC アドレスを基にあて先を判断する

データリンク層

  • 直接接続されている機器のやり取り

物理層

  • 電気的な接続
  • 機器
    • ハブ / リピータハブ
      • コリジョンドメイン
        • 分割しない
      • ブロードキャストドメイン
        • 分割しない
      • 特徴
        • 受信ポート以外の全ポートからデータを送信する(あて先を判断しない)
    • リピータ
      • 伝送途中で減衰した信号レベルの補正と再生増幅を行なう
