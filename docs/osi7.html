OSI 基本参照モデル 7 階層モデル

  • OSI(Open Systems Interconnection : 開放型システム間相互接続)
  • 現在は使われてないけど、参照モデルとして残っている
  • 国際標準化機構である ISO(International Organization for Standardization) が制定したもの

7. アプリケーション層

  • アプリケーションのやり取り
  • プロトコル
    • HTTP
    • SMTP

6. プレゼンテーション層

  • データの表現方法
  • エンディアン

5. セッション層

  • 通信の開始と終了の手順
  • TCP or UDP どちらを利用するかの選択

4. トランスポート層

  • ネットワークの通信管理
  • ネットワーク層から渡されたデータをアプリケーションにきちんと渡したり、両端のコンピュータ間で行われるデータ転送に信頼性を提供したりするための処理
  • プロトコル
    • TCP
    • UDP
  • 機器
    • ゲートウェイ(L4 以上)
      • 異なったプロトコル体系のネットワーク間の接続などに用いられる
    • デフォルトゲートウェイ
      • 個人 PC にはルータの設定が記載
      • 異なるネットワーク帯に接続
    • プロキシサーバ

3. ネットワーク層

  • ネットワークにおける通信経路の選択
  • プロトコル
    • ICMP
  • 機器
    • ルータ

2. データリンク層

  • 直接接続されている機器のやり取り
  • 機器
    • ブリッジ
      • コリジョンドメイン
        • 分割する
      • ブロードキャストドメイン
        • 分割しない
      • 特徴
        • フレームフィルタリング機能
        • MAC アドレスを基にあて先を判断する
    • スイッチングハブ
      • 特徴
        • フレームフィルタリング機能
        • MAC アドレスを基にあて先を判断する
          • => 内部メモリにキャッシュする
          • => まだ学習してない MAC アドレスあてのフレームやブロードキャストはすべてのポートから送信される

1. 物理層

  • 電気的な接続
  • 機器
    • ハブ / リピータハブ
      • コリジョンドメイン
        • 分割しない
      • ブロードキャストドメイン
        • 分割しない
      • 特徴
        • 受信ポート以外の全ポートからデータを送信する(あて先を判断しない)
    • リピータ
      • 伝送途中で減衰した信号レベルの補正と再生増幅を行なう
