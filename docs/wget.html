wget

  • ダウンロードツール

  • 保管場所の指定

    # wget -P 保管ディレクトリ 対象ファイル

  • Basic 認証をこえる

    # wget --http-user=ユーザ名 --http-passwd=パスワード 対象ファイル

  • SSL証明書チェックを行わない

    # wget --no-check-certificate 対象ファイル

  • コンテンツに記載していあるリンクを再帰的に取得

    # wget --recursive http://example.com/index.html

  • Options

    • --level 1
      • 1 階層のみリンクを再帰的に取得
    • --spider
      • ファイルをダウンロードしない
      • HEAD リクエスト実施
