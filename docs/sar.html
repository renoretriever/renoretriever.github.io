sar

  • 一定間隔で CPU、コンテキストスイッチ、割り込み、ページング、デバイス、メモリ、スワップ、ネットワークなどあらゆる情報を表示

  • sysstat パッケージに含まれている

    • sadc コマンドでシステム状況を収集する
      • /var/log/sa ディレクトリ以下の saXX ファイルに格納される
    • sar コマンドで収集するデータを表示

  • 対象時間を抜き出す

    $ sar -r -s 00:00:00 -e 01:00:00

  • -f : 過去の情報閲覧

    # sar -f /var/log/sa/sa01

  • -b : I/O 転送情報
